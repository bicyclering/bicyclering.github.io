## 货到付款 before of 17:00 

> 1. 待发货 → 货到付款 → 订单数 → 推配货中  → 货到付款 → 订单数
> 
> 2. 订单管理 → 后台管理 → 全部 → 下单时间**(前一天 ~ 当天)** → 搜索 → 导出  → 订单信息
> 
> 3. 筛选 → 订单处理状态 → 配货中**[shfit + 双击] → ctrl + c  → 删除** → 订单支付方式 → 货单付款**[shfit + 双击] → ctrl + c  → 删除** 
> 
> 4. 最终数据内容**(配货中的货到付款)**
> 
> 5. WMS → 查物流 → 填写到表格中
> 
> 6. 表格命名方式: 上元堂官网_中通货到付款_年/月/日
